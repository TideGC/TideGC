## 常见目录

Linux 的软件安装目录是也是有讲究的，理解这一点，在对系统管理是有益的

| 目录 | 说明 | 类比 Windows |
| :- | :- | :- |
| `/usr` | 系统级的目录 | `C:/Windows/` |
| `/usr/local` | 用户级的程序目录。<br>用户自己编译的软件默认会安装到这个目录下。 | `C:/Progrem Files/`| 
| `/opt` | 用户级的程序目录。一般解压即可用的软件放在这里，不用时，直接 `rm -rf` 掉即可。| `D:/Software` |
