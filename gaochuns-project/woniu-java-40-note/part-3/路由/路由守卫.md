# 路由守卫

路由守卫指的是能够在你进行导航<small>（ 页面切换、调换 ）</small>之前运行某些代码的功能，甚至能去取消将要发生的导航，或将用户导航到其它地方，而非原本要去的哪里。

路由守卫有 3 种：

| # | 说明 |
| -: | :- |
| 1 | 全局路由守卫 |
| 2 | 路由独享守卫 |
| 3 | 组件路由守卫 |
