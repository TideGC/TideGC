## ClassUtils


封装了关于 class 的操作, 比如:


ClassUtils.getShortClassName() : 获取 Class 的名字

ClassUtils.getPackageName() : 获取 class 所在的包名

ClassUtils.getAllSuperclasses() : 获取父类

ClassUtils.getAllInterfaces() : 获取所有接口

Conversion

封装了关于进制的转换操作.
