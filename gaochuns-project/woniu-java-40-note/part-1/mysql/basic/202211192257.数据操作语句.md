`|` 符号表示二选一；`[]` 符号表示可选项；`{}` 符号表示必选项；`<>` 符号表示需要指定。

数据操作语句（DML 语句）用于对表中的数据进行插入（INSERT）、更新（UPDATE）、删除（DELETE）以及合并（MERGE）操作。

#### 插入数据

insert 语句用于插入数据，基本的语法如下：

```mysql
insert into t(col1, col2, ...)
values (val1, val2, ...);

insert into t(col1, col2, ...)
select ...;
```

#### 更新数据

update 语句用于更新表中的数据，基本的语法如下：

```mysql
update <table_name>
set col1 = val1,
col2 = val2,
...
[where condition];
```


#### 删除数据

UPDATE 语句用于更新表中的数据，基本的语法如下：

```mysql
DELETE FROM <table_name>
[WHERE conditions];
```
