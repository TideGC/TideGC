## 问题二解决：Redis 的 SETNX 命令

![](https://woniumd.oss-cn-hangzhou.aliyuncs.com/java/hemiao/E}`0VGD7JSG`GQK{58R2`9H.png)

如果在代码中使用 Redis 的 SETNX 命令，那么使用逻辑的伪代码如下：

```java
String uuid1 = ...;

// lock
set Test uuid1 NX PX 3000

try {
  // biz handle....
} finally {
    // unlock
    String uuid2 = get Test;
    if ( uuid1 == uuid2 ) {
        del Test
    }
}
```
