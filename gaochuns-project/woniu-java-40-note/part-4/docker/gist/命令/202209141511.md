### 导出镜像：docker save 命令

> 了解

使用 `docker save` 命令可将本地镜像库中的镜像导出：

```bash
# 语法：
# docker save <repository>:<tag> -o <repository>.tar

# 例如：
docker save mysql:8.0.26 -o mysql-8.0.28.tar
```

另外，`docker save` 命令支持同时导出多个镜像：

```sh
docker save -o image.tar ubuntu:lasted centos:latest
```

这多个镜像被打进了一个 .tar 文件中。
