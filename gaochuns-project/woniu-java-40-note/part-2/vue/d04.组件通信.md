# 组件通信

通常一个单页应用会以一棵嵌套的组件树的形式来组织：

![component-3](https://woniumd.oss-cn-hangzhou.aliyuncs.com/java/hemiao/20220627135023.png)

- 页面首先分成了顶部导航、左侧内容区、右侧边栏 3 部分

- 左侧内容区又分为上下 2 个组件

- 右侧边栏中又包含了 3 个子组件

各个组件之间以嵌套的关系组合在一起，那么这个时候不可避免的会有组件间通信的需求。
