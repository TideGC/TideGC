## 条件修改：使用 Wrapper

我们“心里”期望执行的 SQL 如下：

``` sql
update
    department
set name     = 'hello-new',
    location = 'world'
where 
    name = 'hello';
```

使用 Wrapper 对应 Java 中的实现如下：

```java
Department department = new Department(null, "hello-new", "world");
Wrapper<Department> wrapper = new QueryWrapper<Department>()
        .eq("name", "hello");

departmentDao.update(department, wrapper);
```

> [!attention] 注意
> 这里的 null 值表示保持原址不变。
