# 颜色相关属性

**color** 属性允许你指定元素中文本的颜色。

有四种方式描述一种颜色：

| # | 方式 |
| :-: | :-|
| RGB 值 | 例如：rgb(100, 100, 190) |
| RGB 十六进制编码 | 例如：#ee3e80 |
| 颜色关键字 |共 147 个。例如：red |
| 色调、饱和度和明度 | 例如：hsl(0, 100%, 100%) |


**background-color** 属性用于设置元素整个盒子（矩形区域）的颜色。

未明确设置该属性时，默认是"透明色" 。

> [!danger] 透明色不是白色！
