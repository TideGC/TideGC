## 插入 html-string 

```dynamic-embed
![[202302271132]]
```

如果你通过插值表达式<small>（"{{  }}"）</small>在页面上插入的内容是一个 HTML 格式字符串，或这是一个包含 HTML 标签的字符串，那么 Vue 默认是会将 HTML 标签进行转义，从而让你**看到标签本身**。例如：

````col
```js 
data() {
  return {
    msg: 'hello <strong>world</strong>'
  }
}
```

```html
<p>msg: {{ msg }}</p>
```
````

但是，你心里的预期是==想让 “\<strong\>” 起到加粗的效果==，而非直接在页面看见它<small>（标签）</small>本身。

那么你需要使用 "v-html" 来代替 \{\{  \}\} 。

``` html
<p>msg: {{msg}}</p>
<p v-html="'msg: ' + msg"></p>
```
