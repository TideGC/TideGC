## Data 的 2 种写法 

```dynamic-embed
![[202302271132]]
```

Vue 的 "data" 属性有 2 种写法，之前的示例全部使用的是第 1 种。

- 写法一（不推荐）：

```js
data: {
  hours: new Date().getHours(),
  msg : 'World',
  dogs: ['tommy', 'jerry', 'ben']
}
 ```

Vue 的 "data" 属性还有第 2 种写法，要求它是一个返回一个对象的函数。

- 写法二（**推荐**）：

```js
data () {
  return {
    hours: new Date().getHours(),
    msg : 'World',
    dogs: ['tommy', 'jerry', 'ben']
  };
}
```

不推荐`写法一`的原因在于：这种写法在未来会出现问题！<small>某些场景（组件）中它不适用。</small>

因此，从这里开始，我们将直接采用`写法二`，不再使用`写法一`。
