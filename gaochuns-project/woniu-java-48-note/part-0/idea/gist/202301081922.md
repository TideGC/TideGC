## 代码提示忽略大小写

idea 代码提示有大小写区分！比如输入 `str` 无法得到提示 String，只有输入 `Str` 才会提示。如何让代码提示忽略大小写呢？

Editor -> General->Code Completion ，把 Match case 前面的 checkbox 取消
